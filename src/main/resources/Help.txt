Commands:
exit - complete the program;
help - help for valid commands;
point (n) - max numbers after the decimal point (Default: 5);
more on/more off - Detailed solution.

Correct characters:
3 + 2 -> 5
3 - 2 -> 1
3 * 2 -> 6
3 / 2 -> 1.50
3 ^ 2 -> 9
32% -> 0.32
180° -> 3.14159
cos60° -> 0.5
sin30° -> 0.5
tg60° -> 1.73205
ctg30° -> 1.73205
